## Install

**Prerequisites**: AFTS is written in [Go](https://golang.org/doc/install). For deployment, [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/) are needed.

Clone the repo:

```bash
git clone git@github.com:mmathys/acfts.git

# or use the ETH Mirror:
git clone git@gitlab.ethz.ch:disco-students/fs20/mmathys-consensus-free-transaction-systems.git
```
